# sampan.me

My personal portfolio built with [Gatsby](https://www.gatsbyjs.org/) and hosted with [Netlify](https://www.netlify.com/). Modified from [bchiang7/v4](https://github.com/bchiang7/v4) whose website can be found here: https://brittanychiang.com. Also uses [Icon Fonts](http://www.onlinewebfonts.com/icon), [Simple Icons](https://simpleicons.org/), and [favicon.io](https://favicon.io/).

## ðŸ›  Installation & Set Up

1. Install the Gatsby CLI

   ```sh
   npm install -g gatsby-cli
   ```

2. Install and use the correct version of Node using [NVM](https://github.com/nvm-sh/nvm)

   ```sh
   nvm install
   ```

3. Install dependencies

   ```sh
   yarn
   ```

4. Start the development server

   ```sh
   npm start
   ```

## ðŸš€ Building and Running for Production

1. Generate a full static production build

   ```sh
   npm run build
   ```

1. Preview the site as it will appear once deployed

   ```sh
   npm run serve
   ```
